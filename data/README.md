# Data

Эта папка содержит стартовые данные для MVP цифрового штурмана аэролодки.

## Важно

Текущие данные являются демонстрационными seed-данными. Они нужны для проектирования, прототипирования и проверки логики маршрута. Их нельзя использовать для реальной навигации.

## Структура

| Папка | Назначение |
| --- | --- |
| `seed` | первичные данные для MVP |
| `sources` | описание внешних источников и плана сбора |
| `raw` | будущие выгрузки из внешних источников без обработки |
| `collected` | будущие обработанные данные |

## Seed-файлы

| Файл | Что содержит |
| --- | --- |
| `seed/boat_profile_raptor_650.json` | профиль аэролодки из постановки и MVP-допущения |
| `seed/surface_costs.json` | коэффициенты поверхностей для расчета маршрута |
| `seed/demo_area.geojson` | демонстрационная область |
| `seed/route_graph.geojson` | стартовый граф маршрутов |
| `seed/hazards.geojson` | стартовые рисковые зоны |
| `seed/boat_state_sample.json` | пример состояния борта |
| `seed/test_scenarios.json` | сценарии проверки MVP |
| `seed/collection_manifest.json` | манифест текущего сбора данных |

## Правило атрибуции

При подключении OpenStreetMap или других внешних данных нужно сохранять источник и лицензионную заметку в манифесте и свойствах объектов.
